Les types de code

Apprenez à coder les différents langage web, Front-End et Back-End.

Front-End

Le Front-End comprend les langages suivants : HTML, CSS et Javascript. Le Front-End est la face visible d'un site internet.

Back-End

Le Back-End comprend les langages suivants : PHP, Sql, Ruby ... . Le Back-End est la face invisble d'un site internet.

Exercices

Les exercices proposés seront des visualisations réelles et concrètes de ce que l'on peut voir sur un site internet.
